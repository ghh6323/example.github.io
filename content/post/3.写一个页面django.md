## 1. Template 模板用法

`render` 用来返回 HTML 模板页面，而不是像 `HttpResponse` 那样返回纯文本。

### 修改前：返回纯文本

```python
from django.shortcuts import render, HttpResponse

def index(request):
    return HttpResponse("欢迎使用")

def user_list(request):
    return HttpResponse("用户列表")

def user_add(request):
    return HttpResponse("添加用户")
```

### 修改后：返回 HTML 模板

```python
from django.shortcuts import render

def index(request):
    return render(request, "index.html")

def user_list(request):
    return render(request, "user_list.html")

def user_add(request):
    return render(request, "user_add.html")
```

> **查找规则**：Django 会根据 `INSTALLED_APPS` 的注册顺序，逐一去每个 app 的 `templates` 目录中查找对应的 HTML 文件。

### 目录结构

在 app 文件夹下新建 `templates` 目录，把相关的 HTML 文件放进去：

```text
project1/
├── app/
│   ├── templates/          ← 新建
│   │   ├── index.html
│   │   ├── user_list.html
│   │   └── user_add.html
│   └── views.py
└── manage.py
```

## 2. Django 静态文件

在项目根目录（`manage.py` 同级）创建 `static` 文件夹：

```text
project1/
├── manage.py
├── project1/
├── app/
└── static/                 ← 新建
    ├── css/
    │   └── style.css       ← 样式文件
    ├── images/
    │   └── cat.png         ← 图片
    └── js/
        └── main.js         ← 功能逻辑
```

### 配置 settings.py

```python
import os

STATIC_URL = '/static/'

STATICFILES_DIRS = [
    os.path.join(BASE_DIR, 'static'),
]
```

### HTML 中引用

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>首页</title>
    <!-- CSS -->
    <link rel="stylesheet" href="/static/css/style.css">
</head>
<body>
    <h1>欢迎使用 Django</h1>

    <!-- 图片 -->
    <img src="/static/images/cat.png" alt="Logo" class="logo">

    <div class="nav">
        <a href="/user/list/">用户列表</a>
        <a href="/user/add/">添加用户</a>
    </div>

    <button class="btn" onclick="sayHello()">点我一下</button>
    <p id="msg"></p>

    <!-- JS -->
    <script src="/static/js/main.js"></script>
</body>
</html>
```

### main.js

```javascript
// 点击按钮弹出提示
function sayHello() {
    alert("你好，欢迎使用 Django！");
    document.getElementById("msg").innerText = "按钮已点击 ✓";
}

// 页面加载完成后在控制台打印
console.log("页面加载完成，main.js 正常运行");
```

## 3. Django 模板语法

### 核心概念

| 术语 | 含义 | 示例 |
|------|------|------|
| **key（键）** | 字典中的"名字"，用于查找 | `"name"`、`"age"` |
| **value（值）** | 字典中对应的数据内容 | `"Dut"`、`20` |
| **item（项）** | 一个完整的 key-value 对 | `("name", "Dut")` |

```text
字典:  {"name": "Dut", "age": 20}
        └─┬─┘  └─┬─┘   └┬┘  └┬┘
          │      │      │    │
         key   value   key  value

item:   ("name", "Dut")      ("age", 20)
        └──────┬──────┘      └────┬────┘
               │                  │
            一个 item          一个 item
```

### 变量渲染

**views.py：**

```python
def tpl(request):
    name = "张三"
    roles = ["管理员", "CEO", "保安"]
    user_info = {"name": "李四", "salary": 100000, "role": "CTO"}

    return render(request, "tpl.html", {
        "n1": name,
        "n2": roles,
        "n3": user_info
    })
```

**tpl.html：**

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>模板语法</title>
</head>
<body>
    <h1>模板语法的学习</h1>

    <!-- 单个变量 -->
    <div>{{ n1 }}</div>                    <!-- 输出：张三 -->

    <!-- 列表整体 -->
    <div>{{ n2 }}</div>                    <!-- 输出：['管理员', 'CEO', '保安'] -->

    <!-- 列表索引取值 -->
    <div>{{ n2.0 }}</div>                  <!-- 输出：管理员 -->
    <div>{{ n2.1 }}</div>                  <!-- 输出：CEO -->
    <div>{{ n2.2 }}</div>                  <!-- 输出：保安 -->

    <!-- 列表循环 -->
    <div>
        {% for item in n2 %}
            <span>{{ item }}</span>
        {% endfor %}
    </div>
    <!-- 输出：管理员 CEO 保安 -->

    <!-- 字典 key 取值 -->
    <div>{{ n3.name }}</div>                <!-- 输出：李四 -->

    <!-- 字典遍历 values -->
    <div>
        {% for item in n3.values %}
            <span>{{ item }}</span>
        {% endfor %}
    </div>
    <!-- 输出：李四 100000 CTO -->

    <!-- 字典遍历 keys -->
    <div>
        {% for item in n3.keys %}
            <span>{{ item }}</span>
        {% endfor %}
    </div>
    <!-- 输出：name salary role -->

    <!-- 字典遍历 items（key + value） -->
    <div>
        {% for key, value in n3.items %}
            <span>{{ key }}: {{ value }}</span>
        {% endfor %}
    </div>
    <!-- 输出：name: 李四 salary: 100000 role: CTO -->
</body>
</html>
```

### 模板语法速查表

| 语法 | 作用 |
|------|------|
| `{{ 变量 }}` | 输出变量值 |
| `{{ 列表.0 }}` | 列表索引取值（从 0 开始） |
| `{{ 字典.key }}` | 字典 key 取值 |
| `{% for item in 列表 %}` | 循环遍历列表 |
| `{% for key, value in 字典.items %}` | 循环遍历字典 |
| `{% if 条件 %}` | 条件判断 |

## 4. 视图函数 render 的内部流程

```text
用户请求
    ↓
views.py 调用 render(request, "tpl.html", {"n1": name, "n2": roles})
    ↓
1. 读取含有模板语法的 HTML 文件
    ↓
2. 内部进行渲染（模板语法执行并替换数据）
    ↓
3. 最终得到只包含 HTML 标签的字符串
    ↓
4. 将渲染完成的字符串返回给用户浏览器
```

## 5. Requests 库（第三方 HTTP 请求）

`requests` 是 Python 最常用的 HTTP 请求库，用来发送网络请求（GET、POST 等）获取数据。

### 安装

```bash
pip install requests
```

### 在 Django 中使用

**views.py：**

```python
import requests
from django.shortcuts import render

def weather(request):
    # 调用第三方天气 API
    url = "https://api.example.com/weather"
    params = {"city": "上海"}

    response = requests.get(url, params=params)
    data = response.json()  # 把 JSON 转成 Python 字典

    # 提取需要的数据
    weather_info = {
        "city": data["city"],
        "temp": data["temperature"],
        "desc": data["description"]
    }

    # 传给模板渲染
    return render(request, "weather.html", {"weather": weather_info})
```

**weather.html：**

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>天气</title>
</head>
<body>
    <h1>{{ weather.city }} 天气</h1>
    <div>温度：{{ weather.temp }}°C</div>
    <div>天气：{{ weather.desc }}</div>
</body>
</html>
```

### 常用方法

| 方法 | 说明 |
|------|------|
| `requests.get(url, params={})` | GET 请求，获取数据 |
| `requests.post(url, data={})` | POST 请求，提交数据 |
| `response.status_code` | 状态码（200 成功） |
| `response.json()` | 返回 JSON 数据（字典） |
| `response.text` | 返回文本内容 |

### 前后端数据流向

```text
用户浏览器（前端）              第三方 API 服务器
     │                              │
     │  ① 访问 /weather/            │
     └────────→ Django 后端        │
              views.py             │
              requests.get() ─────→│
              获取数据 ←───────────┘
              render 渲染模板
     ←──────── 返回 HTML 页面
```

---

> **注意**：`requests` 是 Python 后端库，HTML 前端发请求要用 JavaScript 的 `fetch()` 或 `axios`。
